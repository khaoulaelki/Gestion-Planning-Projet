# Smart Project Management System

## Description
A project management application developed using Java Swing (MVC architecture) with MySQL database integration.

## Features
- User authentication (Manager / Member)
- Member management
- Project management
- Task management
- Project tracking dashboard
- MySQL database connectivity

## Technologies Used
- Java
- Java Swing
- MySQL
- JDBC
- MVC Architecture

## Installation

1. Clone the repository:
        git clone https://github.com/khaoulaelki/Gestion-Planning-Projet




2. Open the project in your preferred IDE.

3. Run the `LoginView.java` file to start the application.
